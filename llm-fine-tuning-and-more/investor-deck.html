the judge: SCM

"deterministic clinical assessments—auditable and safe"

why deterministic, not neural?

  • clinical assessments must be auditable
  • same input → same output
  • safety-critical = not probabilistic

components

  1. forced alignment (Montreal FA)
  2. weighted phonological distance
  3. error classification
  4. disorder classification
  5. developmental norms DB

typeexample

real-time workflow: the articulation loop

child says "wabbit" → system responds in ~1.5 seconds

1. handoff — ray app sends audio + target
↓ 300ms
2. SAM (ears) — /w/ /a/ /b/ /i/ /t/ (99% conf)
↓ 100ms
3. SCM (judge) — substitution /r/→/w/, severity 0.3
↓ 50ms
4. policy — fail=3, "scaffold, use metaphor"
↓ 800ms
5. SRM (brain) — "let's play! growl like a tiger! GRRRR!"
↓ 300ms
6. SSM (voice) — audio + tiger animation delivered

total latency: ~1.5s
